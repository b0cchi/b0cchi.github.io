‘/etc/mysql/conf.d/my.cnf’ is ignored.の対処法

 2022-06-03  207 words  1 minute  docker mysql 

DockerでMySQLのコンテナを立ち上げた際に、以下のエラーが出た場合の対処法です。

mysql: [Warning] World-writable config file ‘/etc/mysql/conf.d/my.cnf’ is ignored.

恐らくこのエラーが表示されるのはWindows環境だけで，ファイル権限が原因で発生するエラーのようです。

このエラーの対処としてやることは１つだけです。
読み込ませたいmy.cnfを右クリックして読み取り専用(R)にチェックを入れるだけです。

よいコンテナ体験を！！
